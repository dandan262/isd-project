\# isdproject — Requirements



\## Functional Requirements

\- F01: Sistem dapat menginisialisasi project Next.js dengan Tailwind CSS dan Shadcn UI.

\- F02: Sistem terintegrasi dengan PostgreSQL (Supabase) via Prisma ORM.

\- F03: Sistem menyediakan skema database untuk User, Conversation, dan Message.

\- F04: Sistem terintegrasi dengan Gemini API via `@google/generative-ai`.

\- F05: Sistem memiliki endpoint `/api/chat` untuk memproses riwayat chat.

\- F06: UI menyediakan antarmuka chat dengan fitur streaming teks.

\- F07: Sistem mampu menangani upload file (PDF/TXT) untuk dijadikan konteks tambahan bagi Gemini.
