\# isdproject — Vision and Scope



\## Problem Statement

Pengguna membutuhkan asisten AI yang cerdas, responsif, memiliki memori chat, serta mampu memahami konteks dokumen tanpa kebocoran data API key.



\## Scope (MVP)

\- Integrasi Gemini API dengan Streaming Response.

\- Manajemen Sesi \& Riwayat Chat.

\- Context Management (Penyuntikan Dokumen).

\- Autentikasi Pengguna (Google Auth / Email).



\## Tech Stack \& Architecture

\- Frontend \& Backend: Next.js / Node.js (TypeScript)

\- Database: PostgreSQL (Supabase / Vector Enabled)

\- AI Engine: Gemini API via Backend Routes



\## Out of Scope

\- Sistem pembayaran (Subscription billing).

\- Aplikasi Mobile Native.
