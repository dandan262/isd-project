\# Project Configuration



\## Tech Stack

\- \*\*Frontend + Backend\*\*: Next.js (React + Node.js)

\- \*\*Database\*\*: PostgreSQL via Docker

\- \*\*ORM\*\*: Prisma

\- \*\*Auth\*\*: NextAuth.js / Built-in

\- \*\*Deployment\*\*: Vercel



\## Why This Stack

\- Tim nyaman menggunakan ekosistem React dan TypeScript end-to-end.

\- File-based routing di Next.js memudahkan manajemen halaman.

\- Prisma ORM sangat type-safe dan mempermudah query ke Docker PostgreSQL.

\- Sangat direkomendasikan karena populer dan memiliki support AI Agent yang baik.
